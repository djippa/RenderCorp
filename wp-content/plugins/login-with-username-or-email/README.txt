Allow users to login with Email or Username while logging in to Wordpress.

== Description ==
<p>Allow users to login with either Email Address or Username.</p>
<p>Just install the Plugin, no coding experience required.</p>
 
== Installation ==

There is no setting for this plugin just install it and eveything will work automatically

e.g.

1. Upload `/login-with-username-or-email/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can use Email or username to log in


== Frequently Asked Questions ==

= Will this plugin allow users to login using either their username or their email address? =

YES, the plugin allows either to be used in the login screen.
